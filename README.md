# 哈工大深圳每日上报

### 使用方式

#### 0. 请自行安装selenium以及Chromedriver，参考

https://zhuanlan.zhihu.com/p/373688337

https://www.cnblogs.com/lfri/p/10542797.html

#### 1. 填写用户信息：users.ini
*注意不要加双引号如"abcdefg"，只需要写abcdefg即可*
```
name = xx
account = xxx
password = xxx
```

#### 2. 启动脚本

```python
python3 reporter.py
```
